# Smart-Expense-Tracker-API

#HOW TO RUN THIS PROJECT
1. Install SQLite:
   - Windows: Download sqlite3.dll and sqlite3.h
   - Linux:
       sudo apt install sqlite3 libsqlite3-dev

2. Compile Program:

   g++ main.cpp -o expense_tracker -lsqlite3

3. Run Program:

   ./expense_tracker
